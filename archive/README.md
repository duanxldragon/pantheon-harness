# Archive

This directory stores historical workspace artifacts that are no longer part of the active publishable surface.

Use it for:

- completed rollout task packets
- historical verification evidence
- temporary planning or migration documents kept for traceability

Do not treat `archive/` as part of the default copy set for new repositories.

For new repositories, copy only the maintained distribution surfaces described in [README.md](/D:/workspace/go/pantheon-platform/README.md).
