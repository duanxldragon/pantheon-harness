# Human 适配说明

English version: [human.md](./human.md)

人工工程师同样是 Pantheon Harness Engineering 协议的一等参与者。

## 开始前必读

1. `docs/harness/HARNESS_ENGINEERING_CONTRACT.md`
2. `docs/harness/AGENT_INTERFACE_CONTRACT.md`
3. `docs/harness/TASK_PACKET_SPEC.md`
4. 当前仓库的 `AGENTS.md`

## 人工工作流

对于非琐碎任务：

1. 创建或引用 task packet。
2. 声明主责层与依赖层。
3. 阅读 contract anchors。
4. 做聚焦改动。
5. 运行验证命令。
6. 保存或汇总 evidence。
7. 发起 review。

人工工作和 agent 工作使用同一套完成定义，不存在“人工就可以跳过协议”的例外。
