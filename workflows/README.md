# Workflows

Dynamic workflow patterns based on Claude Dynamic Workflows research.

## Patterns

| Pattern | Description |
|---------|-------------|
| Fan-out and Synthesize | Parallel execution with result aggregation |
| Adversarial Verification | Self-critique and verification loops |
| Generate and Filter | Candidate generation with quality filtering |
| Tournament | Multi-candidate competition |
| Loop Until Done | Iterative refinement |
